(2) 실습 내용 및 수행 방법 
- 실습 내용(목표): 
온라인 쇼핑몰 상품의 이름(상품명) 텍스트를 이용하여, 각 상품이 어떤 카테고리에 속하는지 자동으로 분류하는 인공지능 모델을 구현하는 것이 목표이다.
이를 통해 텍스트 데이터 전처리부터 벡터화, 분류 모델 학습·평가까지의 기본적인 머신러닝 파이프라인을 경험하고, 실제 전자상거래 서비스에서 상품 분류 자동화가 어떻게 활용될 수 있는지 이해하고자 했다.
- 수행 방법 및 과정을 기술,
1. Colab 노트북에서 제공된 상품 데이터셋을 불러와 상품명과 정답 카테고리(레이블) 컬럼을 확인하고, 결측치 여부 및 각 카테고리의 데이터 개수를 간단히 탐색하였다.
2. 상품명 텍스트를 소문자로 변환하고, 불필요한 특수문자 및 숫자를 제거하는 전처리를 수행한 뒤, 띄어쓰기를 기준으로 토큰화하여 단어 단위로 나누었다.
3. 전처리된 텍스트에 대해 TF-IDF(Vectorizer)를 적용하여, 각 상품명을 고차원 수치 벡터로 변환하였다.
4. 학습용(train)과 평가용(test) 데이터로 나누어, 학습 데이터에 대해 지도학습 기반의 분류 모델(예: 로지스틱 회귀 또는 나이브 베이즈)을 학습시켰다.
5. 테스트 데이터에 대해 예측을 수행하고, 실제 정답과 비교하여 정확도(accuracy)를 계산하였다.
6.  실제 서비스 상황을 가정하여, 과제에서 제시한 15개 테스트 상품명과 내가 임의로 입력한 실제 판매용 상품명을 모델에 넣어 분류 결과를 확인하고, 어떤 경우에 잘 맞고 어떤 경우에 오분류가 발생하는지 분석하였다.
