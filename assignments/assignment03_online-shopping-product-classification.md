(1) 실습 내용 및 수행 방법 
- 실습 내용(목표): 
온라인 쇼핑몰 상품의 이름(상품명) 텍스트를 이용하여, 각 상품이 어떤 카테고리에 속하는지 자동으로 분류하는 인공지능 모델을 구현하는 것이 목표이다.
이를 통해 텍스트 데이터 전처리부터 벡터화, 분류 모델 학습·평가까지의 기본적인 머신러닝 파이프라인을 경험하고, 실제 전자상거래 서비스에서 상품 분류 자동화가 어떻게 활용될 수 있는지 이해하고자 했다.
- 수행 방법 및 과정을 기술,
1. Colab 노트북에서 제공된 상품 데이터셋을 불러와 상품명과 정답 카테고리(레이블) 컬럼을 확인하고, 결측치 여부 및 각 카테고리의 데이터 개수를 간단히 탐색하였다.
2. 상품명 텍스트를 소문자로 변환하고, 불필요한 특수문자 및 숫자를 제거하는 전처리를 수행한 뒤, 띄어쓰기를 기준으로 토큰화하여 단어 단위로 나누었다.
3. 전처리된 텍스트에 대해 TF-IDF(Vectorizer)를 적용하여, 각 상품명을 고차원 수치 벡터로 변환하였다.
4. 학습용(train)과 평가용(test) 데이터로 나누어, 학습 데이터에 대해 지도학습 기반의 분류 모델(예: 로지스틱 회귀 또는 나이브 베이즈)을 학습시켰다.
5. 테스트 데이터에 대해 예측을 수행하고, 실제 정답과 비교하여 정확도(accuracy)를 계산하였다.
6.  실제 서비스 상황을 가정하여, 과제에서 제시한 15개 테스트 상품명과 내가 임의로 입력한 실제 판매용 상품명을 모델에 넣어 분류 결과를 확인하고, 어떤 경우에 잘 맞고 어떤 경우에 오분류가 발생하는지 분석하였다.

- 실습 결과 설명,
1) 최종 테스트세트 15개 상품 분류를 잘했나요?
 (오류개수: 1개)
학습/훈련 단계에서 확인한 전체 테스트 정확도는 약 0.86이었고, 별도로 제시된 15개 상품에 대해서는 1개만 오분류되어 약 93%로 비교적 높게 나왔다. 최종 테스트 세트 15개를 따로 넣어본 결과에서도 1개의 오분류가 있었지만 대부분은 올바르게 분류되어,전체적인 성능 수준은 학습 단계에서 확인한 테스트 정확도와 크게 다르지 않은 것으로 보였다.
다만 실제 서비스에 적용하기에는 아직 오분류가 존재하고, 특히 사람 눈으로 보면 헷갈리지 않는 상품인데도 모델이 잘못 예측하는 경우가 있어서 추가적인 학습과 성능 개선이 더 필요하다고 판단하였다. 수업 과제 수준에서는 모델이 기본적인 패턴을 잘 학습했다고 볼 수 있지만, 실제 쇼핑몰 환경에서 안정적으로 사용하려면 더 많은 데이터와 튜닝 작업이 요구된다.
오류를 줄이기 위해서는 학습 데이터의 양과 다양성을 늘리거나, 비슷한 상품 간 특징을 더 잘 구분할 수 있도록 모델 구조를 개선하고 하이퍼파라미터를 조정하는 방법이 있다. 또한 잘못 분류된 이미지들만 모아서 별도로 재학습시키거나, 데이터 증강(augmentation) 기법을 적용해 다양한 형태의 입력에 대해 더 강건한 모델을 만드는 방식으로 오류를 점진적으로 보완할 수 있을 것이다.

2) 마지막으로 입력한 진짜 판매를 위한 상품은 무엇이었나요?
마지막으로 입력한 실제 판매용 상품은 Ankle Boots였다.  
이 상품에 대해 모델은 ‘Ankle boot’ 카테고리로 약 93% 확률로 분류했으며, 이는 내가 의도한 실제 카테고리와도 일치하는 결과였다.  
이를 통해 모델이 부츠 계열 상품의 특징을 비교적 잘 학습하고 있다는 점을 확인할 수 있었다.


(3) 실습 결과 분석 
- 실습에 적용된 인공지능 원리,
이번 실습은 상품명이라는 텍스트 데이터를 입력으로 받아, 미리 정의된 여러 카테고리 중 하나를 예측하는 지도학습 기반의 다중 분류(multi-class classification) 문제로 구성되었다.
텍스트를 컴퓨터가 처리할 수 있도록 숫자 벡터로 변환하기 위해 TF-IDF와 같은 벡터화 기법을 사용하였고, 이렇게 만들어진 피처를 입력으로 받아 각 카테고리의 소속 확률을 계산하는 선형 분류 알고리즘(예: 로지스틱 회귀)을 학습시켰다.
이 과정에서 “텍스트 전처리 → 벡터화 → 모델 학습 → 평가”로 이어지는 전형적인 머신러닝 텍스트 분류 파이프라인의 구조를 실제로 경험할 수 있었다.

- 본인 전공에서의 활용 방안,
정보통신공학 관점에서, 이번 실습은 서비스 데이터(상품명 텍스트)를 이용해 실제 서비스 기능(상품 분류)을 자동화하는 전형적인 응용 사례를 보여준다. 전공에서 배우는 네트워크·시스템 기술 위에, 이런 머신러닝 모델을 덧붙이면 대규모 온라인 쇼핑몰이나 플랫폼에서 상품 등록 자동화, 검색 품질 개선, 추천 시스템 고도화 등을 구현할 수 있다.
예를 들어, 판매자가 상품명을 대충 입력하더라도 모델이 자동으로 카테고리와 태그를 제안해주면, 운영 인력의 부담을 줄이고 데이터 구조를 더 깔끔하게 유지할 수 있다. 이렇게 정리된 구조화 데이터는 이후 매출 분석, 클릭률 분석, A/B 테스트 등 다양한 데이터사이언스 작업의 기반이 된다.
또한, 잘 분류된 카테고리 정보를 활용해 “어떤 상품군이 성장 잠재력이 높은지”, “어떤 카테고리가 자주 함께 조회되는지”를 분석하면, 마케팅·프로모션 전략이나 추천 알고리즘 설계에도 직접적으로 활용할 수 있다.
이번 실습 경험은 단순히 모델 정확도만 보는 것을 넘어, 인공지능 모델이 실제 비즈니스 로직과 어떻게 연결되고, 전공 지식(네트워크·플랫폼 개발)과 결합되어 어떤 서비스를 만들 수 있는지를 상상해 보게 해주었다.
