# [08주차] 자율지능과 인공지능

## 1. 오늘의 키워드
- 오토파일럿 기술
- 자율주행차, 자율주행 관련 핵심 기술
- 자율주행과 인공지능 기술
- 자율주행차의 기술적 한계, 법적 책임, 윤리적 문제
- 트롤리 딜레마(Trolley Dilemma)

## 2. 강의 핵심 내용 요약
- 먼저 오토파일럿 기술과 자율주행차 개념을 소개하며, 사람이 직접 조작하지 않아도 차량이 주변 환경을 인식하고 주행 경로를 결정하는 기술이 점점 상용화 단계에 들어서고 있다는 점을 설명했다.  
- 자율주행차 관련 기술로는 센서(카메라, 레이더, 라이다 등), 위치 파악(GPS), 지도 및 경로 계획, 차량 제어 시스템, 그리고 이 모든 정보를 통합해 판단을 내리는 인공지능 알고리즘이 언급되었다.  
- 자율주행과 인공지능 기술의 관계에서는, 이미지·영상 인식을 통해 차선, 보행자, 신호등, 장애물을 구분하고, 이를 바탕으로 경로를 예측·계획하는 데 딥러닝 기반 컴퓨터 비전 기술이 핵심 역할을 한다는 점을 강조했다.  
- 인공지능 핵심 기술 측면에서는 대규모 데이터(도로 상황, 교통 패턴), 딥러닝 모델(특히 CNN 기반 시각 인식 모델), 시뮬레이션 환경, 실시간 의사결정 시스템 등이 자율주행의 토대가 된다고 설명했다.  
- 자율주행 자동차의 해결 과제로는  
  - 센서 오작동, 악천후, 예외적인 상황에 대한 대응이 어려운 등 **기술적 한계**,  
  - 사고 발생 시 제조사, 소프트웨어 개발자, 운전자 중 누구에게 책임을 물을 것인지에 대한 **법적 책임 문제**,  
  - 피할 수 없는 사고 상황에서 누구를 우선 보호할 것인지와 같은 **윤리적 문제(트롤리 딜레마)** 가 제시되었다.  
- 트롤리 딜레마 사례를 통해, 인간 운전자에게도 어려운 선택을 기계에게 맡길 때 생기는 윤리적·철학적 질문과, 이를 코드·알고리즘으로 어떻게 표현할 수 있을지에 대한 고민이 필요하다는 점을 다루었다.  
- 마지막으로 자율실습에서는 이미지 인식 인공지능 학습을 통해, 자율주행차가 도로 환경을 어떻게 인식하고 분류하는지의 기초 개념을 체험하는 시간을 가졌다.

## 3. 인상 깊었던 포인트
- 오토파일럿이나 자율주행을 단순히 '차가 자동으로 간다' 수준으로 생각했는데, 실제로는 센서·지도·통신·인공지능이 모두 연결된 복잡한 시스템이라는 점이 인상 깊었다.  
- 자율주행에 필요한 인공지능 기술이 단순히 이미지 인식 하나가 아니라, 주변 상황을 종합적으로 이해하고 미래를 예측하는 일종의 '종합 의사결정 시스템'이라는 설명이 기억에 남았다.  
- 기술적 한계뿐 아니라, 사고가 났을 때 책임을 누가 져야 하는지, 피할 수 없는 사고 상황에서 어떤 선택을 해야 하는지 같은 법적·윤리적 문제가 기술 발전만큼이나 중요하다는 점이 새롭게 느껴졌다.  
- 트롤리 딜레마처럼 원래는 철학 수업에서 다루던 주제가, 이제는 실제 자율주행 알고리즘 설계와 연결된 현실적인 고민이라는 사실이 조금 무섭기도 하고 인상적이었다.

## 4. 내 생각 / 느낀 점
- 자율주행차가 상용화되면 단순히 운전이 편해지는 수준을 넘어, 물류·교통·도시 구조 자체가 바뀔 수 있다는 생각이 들었다. 동시에, 이런 변화의 기반에는 결국 방대한 주행 데이터와 그 데이터를 학습한 인공지능 모델이 있다는 점이 흥미로웠다.  
- 데이터사이언티스트를 목표로 할 때, 자율주행 분야는 단순 '멋있는 기술'이 아니라, 실제 인간의 생명과 직결되는 의사결정에 관여하는 데이터·모델을 다루는 영역이라는 점에서 더 큰 책임감이 필요하다고 느꼈다.  
- 기술적 정확도뿐 아니라 윤리적 기준, 법적 책임 범위까지 고려해 모델을 설계해야 한다는 점에서, 앞으로의 인공지능 개발은 여러 분야(법, 윤리, 정책)와 협력하는 형태가 될 것 같다고 생각했다.  
- 이미지 인식 인공지능 학습 실습을 통해, 자율주행차에서 쓰이는 컴퓨터 비전 기술의 아주 기초를 경험해볼 수 있었고, 나중에 여유가 되면 실제 도로 이미지 데이터로 간단한 차선·표지판 분류 모델을 만들어 보고 싶다는 생각이 들었다.

## 5. 더 찾아보고 싶은 것 (데이터사이언티스트 관점)
- 공개된 자율주행 관련 데이터셋(예: 도로 영상, 보행자·차선·표지판 라벨링 데이터)을 활용해, 간단한 이미지 분류나 객체 탐지 모델을 직접 만들어보는 미니 프로젝트를 해보고 싶다.  
- 자율주행 사고 사례에서 어떤 데이터·상황이 문제였는지 분석한 리포트들을 찾아보고, 데이터 수집·전처리·테스트 단계에서 어떤 보완이 가능했는지 데이터사이언티스트 입장에서 정리해보고 싶다.  
- 자율주행 의사결정 로직에 트롤리 딜레마 같은 윤리적 기준을 실제로 반영하려는 시도가 있는지, 그리고 이를 수학적·알고리즘적으로 어떻게 표현하려 하는지 연구 사례를 찾아보고 싶다.  
- 자율주행 시스템 전체에서 센서 데이터 처리, 모델 추론, 로그 분석, 시뮬레이션 테스트까지 이어지는 데이터 파이프라인 구조를 공부해서, 장기적으로는 이런 시스템에서 데이터사이언티스트가 맡는 역할을 구체적으로 상상해 보고 싶다.
