# [07주차] 인공지능 기술의 발전

## 1. 오늘의 키워드
- 생활 속 인공지능(추천, 음성인식, 얼굴인식 등)
- 비지도학습(Unsupervised Learning)
- 지도학습(Supervised Learning)
- 인공지능 활용 분야(이미지·음성·자연어·추천 시스템 등)
- 인공지능의 미래, 사람의 두뇌와 인공지능

## 2. 강의 핵심 내용 요약
- **'나를 알아보는 핸드폰'** 예시를 통해, 스마트폰 잠금 해제에 쓰이는 얼굴 인식·지문 인식 등이 인공지능 기술과 밀접하게 연결되어 있으며 사용자의 패턴을 학습해 점점 더 정교해진다는 점을 설명했다.  
- 비지도학습(unsupervised learning)은 정답 레이블이 없는 데이터를 스스로 묶거나 구조를 찾는 방식으로, 예를 들어 비슷한 취향의 사용자를 군집화하거나 비슷한 상품을 자동으로 묶어내는 추천의 기초가 될 수 있음을 소개했다.  
- 지도학습(supervised learning)은 입력과 정답 레이블이 함께 주어진 데이터를 학습해 새 입력에 대한 예측을 수행하는 방식이며, 이미지 분류, 스팸 메일 분류, 음성 인식, 번역 등 실생활 서비스의 핵심 기술로 쓰이고 있음을 설명했다.  
- 인공지능 활용 분야로는 이미지·음성·자연어 처리, 게임, 자율주행, 금융·헬스케어·제조·교육 등 매우 다양한 영역이 소개되었고, 특히 우리가 매일 사용하는 검색, 추천, 광고, SNS 피드 정렬 등도 모두 인공지능 기술 위에 올라가 있다는 점을 강조했다.  
- '인공지능을 사람의 두뇌와 연결한다면?'이라는 질문을 던지며 뇌-컴퓨터 인터페이스(BCI)나 뉴럴링크(Neuralink)와 같은 시도들을 간단히 소개하고, 기술 발전과 윤리·철학적 고민이 함께 필요하다는 점을 언급했다.  
- Python 자율 실습을 통해, 간단한 데이터 구조와 제어문, 라이브러리 사용법 등을 연습하면서 실제로 인공지능·데이터 분석 프로젝트에서 가장 많이 쓰이는 언어가 파이썬이라는 점을 체감하게 해 주었다.

## 3. 인상 깊었던 포인트
- '나를 알아보는 핸드폰'이라는 표현이 직관적이어서, 잠금 해제나 사진 앨범의 얼굴 분류 같은 기능도 결국은 **데이터를 통해 나를 학습하는 인공지능 모델**이라는 점이 인상 깊었다.  
- 비지도학습과 지도학습이 단순한 개념 설명이 아니라, 실제 서비스(추천 시스템, 스팸 필터, 이미지 분류 등) 예시와 연결돼서 설명되니까 머릿속에 훨씬 잘 남았다.  
- 인공지능이 이제는 하나의 독립된 기술이 아니라, 검색·쇼핑·콘텐츠 소비·SNS 등 **일상 대부분의 서비스에 자연스럽게 녹아 있는 인프라**처럼 느껴진다는 점이 새롭게 다가왔다.  
- 사람의 두뇌와 인공지능을 직접 연결하려는 시도가 있다는 이야기를 들으면서, 기술의 가능성과 동시에 '어디까지가 인간이고 어디서부터가 기계인가'라는 철학적인 질문도 떠올랐다.

## 4. 내 생각 / 느낀 점
- 인공지능 기술이 발전하면서, 내가 매일 쓰는 서비스들(유튜브 추천, 음악 스트리밍, 온라인 쇼핑 등)이 사실은 모두 **데이터와 모델이 뒤에서 돌고 있는 결과**라는 게 더 선명하게 느껴졌다.  
- 데이터사이언티스트를 목표로 할 때, 단순히 모델을 만드는 기술뿐 아니라 '어떤 데이터를 어떻게 모으고, 어떤 피처를 쓰고, 어떤 피드백으로 모델을 개선할지'를 설계하는 능력이 중요하다는 생각이 들었다.  
- Python 실습을 통해 기본 문법을 다루면서, 앞으로 실제로 데이터를 가져와서 전처리하고 간단한 모델을 만들어보는 단계까지 빠르게 나아가 보고 싶다는 욕심이 생겼다.  
- 인공지능을 사람의 두뇌와 직접 연결하는 기술은 아직 멀게 느껴지지만, 헬스케어·보조공학 같은 분야에서는 분명히 긍정적 영향이 있을 수 있어 기술 발전과 윤리적 기준이 함께 논의되어야 한다고 느꼈다.

## 5. 더 찾아보고 싶은 것 (데이터사이언티스트 관점)
- 내 스마트폰 사용 데이터(스크린 타임, 앱 사용 패턴 등)를 익명·개인 용도로 수집해, 간단한 **비지도학습(클러스터링)** 으로 하루 패턴이나 앱 사용 유형을 분석해보는 미니 프로젝트를 해보고 싶다.  
- 지도학습과 비지도학습을 모두 활용하는 실제 서비스 사례(예: 추천 시스템에서 군집화 + 랭킹 모델)를 찾아보고, 어떤 데이터 흐름과 구조로 설계되어 있는지 공부해보고 싶다.  
- Python과 scikit-learn, pandas 같은 라이브러리를 활용해 작은 데이터셋으로 **간단한 분류/군집 모델**을 직접 만들어 보고, 수업에서 배운 개념과 코드 구현을 연결해보고 싶다.  
- 뇌-컴퓨터 인터페이스(BCI)처럼 뇌파·생체 신호 데이터를 다루는 분야에서 어떤 데이터 전처리·모델링 기법이 사용되는지, 오픈데이터나 튜토리얼을 찾아보고 장기적으로 도전해보고 싶다.
