# [01주차] 4차 산업혁명의 시작

## 1. 오늘의 키워드
- 4차 산업혁명(초연결, 초지능화)
- 알파고
- 가트너 하이프 사이클

## 2. 강의 핵심 내용 요약
- 1차~3차 산업혁명과 4차 산업혁명의 차이점
- 4차 산업혁명을 이끄는 핵심 기술(AI, IoT, Big Data, Cloud 등)
- 기술 변화가 사회·산업 구조에 미치는 영향
- 앞으로 필요한 역량(데이터 리터러시, 문제 해결 능력 등)에 대한 언급

## 3. 인상 깊었던 포인트
- 인상 깊었던 문장, 사례, 교수님 설명 등 2~4줄
- 예) “4차 산업혁명은 단순한 기술 발전이 아니라, 사회 시스템 전체를 바꾸는 패러다임 전환이다.” 같은 문장 메모

## 4. 내 생각 / 느낀 점
- 이번 주제에 대해 느낀 점, 공감/의문, 내 경험이랑 연결되는 부분
- 예) 군대/학교/알바/서비스 이용하면서 ‘아 이게 4차 산업혁명이구나’ 느꼈던 사례 등
- 데이터사이언티스트 목표랑 어떻게 연결되는지 한두 줄

## 5. 더 찾아보고 싶은 것
- 키워드 2~4개 정도만 적어두기
- 예)
  - 한국에서 4차 산업혁명 관련 실제 정책/사업 사례
  - 데이터 분석 직무와 4차 산업혁명
  - 자동화와 일자리 변화 통계
