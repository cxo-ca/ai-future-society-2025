# [01주차] 4차 산업혁명의 시작

## 1. 오늘의 키워드
- 알파고(AlphaGo)
- 1~3차 산업혁명 vs 4차 산업혁명(초연결, 초지능화)
- 가트너 하이프 사이클(Gartner Hype Cycle)

## 2. 강의 핵심 내용 요약

- 알파고와 이세돌 대국 사례를 통해, 인공지능 기술이 단순 연구를 넘어서 사회적으로 큰 충격과 관심을 불러온 사건이었다는 점을 다루었다.
- 1~3차 산업혁명은 각각 증기기관, 전기에너지 기반의 대량생산, 컴퓨터·인터넷 등장에 의해 촉발되었다. 그리고, 4차 산업혁명은 인공지능과 인터넷 기반의 초연결, 초지능화를 통한 AI혁명이며 지능정보 사회의 시작으로 소개되었다.  
- 가트너 하이프 사이클을 통해 새로운 기술이 등장할 때 기술촉발 → 부풀려진 기대의 정점 → 환멸의 골짜기 → 계몽의 비탈길 → 생산성의 고원 으로 이어지는 패턴을 설명하며, 현재 다양한 인공지능 기술이 어느 구간에 위치하는지 생각해보게 했다.  
- 4차 산업혁명은 단순히 기술 하나의 발전이 아니라, 산업 구조와 일하는 방식, 일자리 형태까지 바꾸는 '패러다임 전환'이라는 점을 강조하였다.

## 3. 인상 깊었던 포인트
- 알파고 사례를 다시 보면서, 바둑처럼 인간의 직관이 중요하다고 여겨졌던 영역도 데이터와 알고리즘으로 어느 정도 대체될 수 있다는 점이 인상깊었다.  
- 가트너 하이프 사이클 그래프가 실제로도 여러 기술(메타버스, 블록체인, 자율주행 등)에 그대로 적용되는 것 같아서, 기술 관련 뉴스를 볼 때 '저 기술은 지금 어느 단계인가?'를 생각해보게 되었다.  
- 4차 산업혁명이 'AI 한 가지 기술의 혁명'이 아니라, 여러 기술이 동시에 발전하고 서로 연결되면서 사회 전체를 바꾼다는 설명이 기억에 남았다.

## 4. 내 생각 / 느낀 점
- 알파고 이후 인공지능이 갑자기 세상을 바꾼 것처럼 느껴졌지만, 실제로는 오랫동안 쌓인 연구와 데이터, 컴퓨팅 자원이 한 번에 터져 나온 결과라는 점이 흥미로웠다.  
- 1~3차 산업혁명과 비교해서, 4차 산업혁명은 변화 속도가 훨씬 빠르고 소프트웨어·데이터 중심이라서, 정보통신공학 전공이 가진 장점이 크겠다는 생각이 들었다.  
- 새로운 기술이 나올 때마다 무조건 혁명적이라고 보는 게 아니라, 가트너 하이프 사이클처럼 일종의 패턴을 가지고 비판적으로 보는 태도가 필요하다는 걸 느꼈다.  
- 앞으로 데이터사이언티스트가 되더라도, 단순히 모델만 만드는 사람이 아니라, 이런 큰 흐름 속에서 어떤 문제를 풀지 정의하는 역할이 중요할 것 같다고 느꼈다.

## 5. 더 찾아보고 싶은 것
- 알파고 이후의 인공지능 발전(알파고 제로, 챗GPT 등)의 차이점  
- 실제 산업 현장에서 말하는 4차 산업혁명 사례(스마트팩토리, 스마트시티 등)  
- 가트너 하이프 사이클에서 내 관심기술이 현재 어느 단계로 평가되는지  
- 4차 산업혁명과 관련된 데이터 분석/AI 직무의 변화와 요구 역량
